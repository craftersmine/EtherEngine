# EtherEngine
###### _make your game..._

-------
New reworked and renamed libraries of GameEngine

Allows you to create games with simple visuals

##### Current Engine Features:
* Full working XInput Gamepad input
* Lightweight
* Content packaging
* Multi-scened
* Texture-based animations
* Camera
